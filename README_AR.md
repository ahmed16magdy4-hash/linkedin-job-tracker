# متتبع وظائف LinkedIn

تطبيق ويب حديث وسهل الاستخدام لتتبع جميع طلبات التوظيف الخاصة بك على LinkedIn في مكان واحد. يوفر لك لوحة تحكم شاملة مع إحصائيات فورية وإدارة كاملة لحالات الطلبات.

## المميزات الرئيسية

- **إضافة وظائف بسهولة**: نموذج سهل الاستخدام لإضافة وظائف جديدة مع جميع التفاصيل
- **تتبع الحالة**: تحديث حالة الطلب بين (تم التقديم، قيد الانتظار، مقابلة، مقبول، مرفوض)
- **إحصائيات فورية**: عرض ملخص شامل لجميع الطلبات بأنواعها المختلفة
- **تصفية ذكية**: تصفية الوظائف حسب الحالة الحالية
- **واجهة عربية**: واجهة مستخدم كاملة باللغة العربية
- **تحديث فوري**: البيانات تتحدث فوراً عند أي تغيير (Real-time)
- **تصميم احترافي**: واجهة حديثة وجميلة مع ألوان احترافية

## المتطلبات

- Node.js 18 أو أحدث
- npm أو pnpm
- حساب Firebase

## التثبيت والتشغيل المحلي

### 1. استنساخ المشروع

```bash
git clone <repository-url>
cd linkedin-job-tracker
```

### 2. تثبيت المكتبات

```bash
npm install
# أو
pnpm install
```

### 3. إعداد Firebase

اتبع التعليمات الكاملة في ملف [FIREBASE_SETUP.md](./FIREBASE_SETUP.md)

### 4. تشغيل التطبيق محلياً

```bash
npm run dev
# أو
pnpm dev
```

سيفتح التطبيق على `http://localhost:3000`

## الاستخدام

### إضافة وظيفة جديدة

1. انقر على زر **"إضافة وظيفة"** في الأعلى
2. ملء النموذج بالمعلومات التالية:
   - **اسم الشركة** (مطلوب)
   - **اسم الوظيفة** (مطلوب)
   - **رابط الوظيفة** (اختياري)
   - **الموقع** (اختياري)
   - **الراتب** (اختياري)
   - **الحالة** (الافتراضي: تم التقديم)
   - **ملاحظات** (اختياري)
3. انقر على **"إضافة الوظيفة"**

### تحديث حالة الوظيفة

1. انقر على بطاقة الوظيفة
2. اختر الحالة الجديدة من الأزرار الموجودة على البطاقة
3. ستتحدث الحالة فوراً

### حذف وظيفة

1. انقر على رمز سلة المهملات في بطاقة الوظيفة
2. أكد الحذف في النافذة المنبثقة

### تصفية الوظائف

استخدم الأزرار في الأعلى لتصفية الوظائف حسب الحالة:
- **الكل**: عرض جميع الوظائف
- **تم التقديم**: الوظائف التي تم التقديم عليها للتو
- **قيد الانتظار**: الوظائف التي تنتظر الرد
- **مقابلات**: الوظائف التي وصلت لمرحلة المقابلة
- **مقبول**: الوظائف المقبولة
- **مرفوض**: الوظائف المرفوضة

## استكشاف الأخطاء

### البيانات لا تحفظ

- تأكد من أن Firebase مهيأ بشكل صحيح
- تأكد من تفعيل Firestore Database
- افتح أدوات المطور (F12) وتحقق من الأخطاء

### الواجهة لا تظهر بشكل صحيح

- امسح ذاكرة التخزين المؤقت (Cache): Ctrl+Shift+Delete
- تأكد من تثبيت جميع المكتبات: `npm install`
- أعد تشغيل خادم التطوير: `npm run dev`

### مشاكل الأداء

- استخدم أدوات المطور لفحص الأداء
- تأكد من عدم وجود استعلامات Firestore غير ضرورية
- قلل عدد الوظائف المعروضة في الصفحة الواحدة
